# Code of Conduct

## Our Pledge

We pledge to make participation in PyTorch Fundamentals Lab respectful, constructive, and welcoming for learners and contributors.

## Expected Behavior

- Be kind and professional.
- Explain technical corrections with enough context to help others learn.
- Keep examples reproducible and distinguish teaching checks from benchmark claims.
- Respect dataset licenses, attribution, and user privacy.

## Unacceptable Behavior

- Harassment, intimidation, or discriminatory language.
- Publishing another person's private information.
- Sharing secrets, restricted datasets, or unsafe serialized model artifacts.
- Presenting fabricated metrics or unverifiable outputs as experimental results.

## Enforcement

Project maintainers may remove comments, close issues, reject pull requests, or block participants whose behavior harms the project community.
